# Support

## Where To Ask For Help

- Use [GitHub Discussions](https://github.com/Elvin-yk/RoboClaw/discussions) for questions, community ideas, and decision discussions
- Use the [Issue Forms](https://github.com/Elvin-yk/RoboClaw/issues/new/choose) for structured proposals
- Use [SECURITY.md](SECURITY.md) for security-related reporting guidance

## Best Channels

- `Q&A`: setup questions, usage questions, and architecture questions
- `Ideas`: early proposals and open-ended suggestions
- `Polls`: community voting on major public decisions
- `Issues`: accepted work, bugs, and structured proposals

If GitHub Discussions is not enabled yet, please open an issue and mention that your post is intended as a community discussion or proposal.
